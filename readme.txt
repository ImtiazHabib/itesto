= Features =

  * Very easy installation
  * Shortcode system
  * Unlimited color variation by using jQuery color picker.
  * Responsive design
  * Flexible and easy to use
  * Lightweight JQuery


== Installation ==
1. Upload `i` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>if(function_exists('itestimo_add_custom_shortcode')){itestimo_loop();}</strong> in template code or  <strong>[itestimo]</strong> in the post/page" to display testimonial.
